Exemple d'attaque XSS

Pour cet exemple d'attaque XSS (Cross-Site Scripting),
les liens sont censés faire passer du texte en information (pour les liens normaux, c'est "bonjour l'été"),
les liens qui tentent une attaque XSS chercher à faire passer du code malicieux (ici, il remplace le lien retour et le titre)

Page vulnérable : Attaque XSS à partir de ce lien Page non vulnérable : Lien normal Page vulnérable : Attaque XSS à partir de ce lien Page vulnérable : Lien normal
